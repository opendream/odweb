# UNICEF LINE RUN

UNICEF LINE Run was a running event for raising funds for disadvantaged children in Thailand. This is the first time participants could purchase event tickets and donate online at the same time through UNICEF’s One Stop Shopping platform supported by Opendream. The event had successfully raised a total amount of ticket sales and donation of 7.9 million baht.
