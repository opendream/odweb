# UNICEF LINE RUN

UNICEF LINE Run งานวิ่งที่จัดขึ้นเพื่อระดมเงินสนับสนุนงานของยูนิเซฟในการพัฒนาความเป็นอยู่ของเด็กทุกคน โดยเฉพาะเด็กๆ ที่ขาดโอกาสที่สุดในสังคม กับครั้งแรกที่เปิดให้ซื้อบัตรและจ่ายเงินออนไลน์ เป็น One Stop Shopping พร้อมด้วยระบบบริจาคเงิน โดยสามารถระดมเงินจากการขายบัตรวิ่งและบริจาคได้ถึง 7.9 ล้านบาท
