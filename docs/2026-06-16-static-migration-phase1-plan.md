# Static Migration Phase 1 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Stand up an Astro site that faithfully renders the WordPress **posts** (TH + EN) with the current site's chrome, fed by a re-runnable REST extraction pipeline, deployed to a Cloudflare Pages preview.

**Architecture:** A Node extraction script reads the local Docker WordPress REST API (`http://localhost:8080/wp-json`), converts each post to a markdown file (+ frontmatter, + downloaded media) under `site/src/content/posts/{th,en}/`. Astro builds those via a content collection, routes them at their original WP paths (TH at `/…`, EN at `/en/…`), and wraps them in components/CSS ported from the live theme. Output is uploaded to a Cloudflare Pages preview with Wrangler.

**Tech Stack:** Astro 5, `@astrojs/sitemap`, Vitest (pipeline unit tests), Turndown (HTML→markdown), Node 20+ (built-in `fetch`), Wrangler (Pages deploy).

**Spec:** `docs/2026-06-16-static-migration-phase1-spec.md`

**Git note:** This workspace is intentionally not under git yet. Treat each **Commit** step as a checkpoint — either initialize a *local* repo in `site/` (Task 1, no remote) and commit, or skip and just verify-and-continue. No pushing.

**Preconditions (verify before starting):**
- Local stack up: `cd local && docker compose ps` shows db/php/web healthy; `curl -s -o /dev/null -w '%{http_code}' http://localhost:8080/` → `200`.
- Node 20+ and `npm` available; `wrangler` available (or `npx wrangler`) with a Cloudflare login for the deploy task only.

- [ ] **Step 3: Implement `site/scripts/lib/convert.mjs`**

```js
import TurndownService from 'turndown';

const UPLOADS_RE = /https?:\/\/[^"')\s]+\/wp-content\/uploads\/([^"')\s]+)/g;

export function urlToPath(link) {
  const { pathname } = new URL(link);
  const p = pathname.replace(/\/+$/, '');
  return p === '' ? '/' : p;
}

// Remove Divi builder wrapper tags (keep their inner content). Conservative:
// only unwraps <div>/<section> whose class mentions et_pb_, leaving content intact.
export function stripDiviCruft(html) {
  return html
    .replace(/<(div|section)\b[^>]*class="[^"]*et_pb_[^"]*"[^>]*>/g, '')
    .replace(/<\/(div|section)>/g, '')      // matching closers (Divi posts are flat enough for P1)
    .replace(/\sdata-[a-z-]+="[^"]*"/g, '')
    .replace(/\n{3,}/g, '\n\n')
    .trim();
}

const turndown = new TurndownService({ headingStyle: 'atx', codeBlockStyle: 'fenced', bulletListMarker: '-' });

export function htmlToMarkdown(html) {
  return turndown.turndown(html).trim();
}

export function extractImageUrls(html) {
  return [...html.matchAll(UPLOADS_RE)].map(m => m[0]);
}

// http://host/wp-content/uploads/2018/09/a.jpg -> /media/2018/09/a.jpg
export function rewriteMediaUrls(html) {
  return html.replace(UPLOADS_RE, (_m, tail) => `/media/${tail}`);
}

function yamlValue(v) {
  if (Array.isArray(v)) return v.length ? '\n' + v.map(x => `  - ${x}`).join('\n') : ' []';
  if (v instanceof Date) return ` ${v.toISOString()}`;
  const s = String(v);
  return /[:#"\n]/.test(s) ? ` "${s.replace(/"/g, '\\"')}"` : ` ${s}`;
}

export function toFrontmatter(data, body) {
  const lines = Object.entries(data)
    .filter(([, v]) => v !== undefined && v !== null)
    .map(([k, v]) => `${k}:${yamlValue(v)}`);
  return `---\n${lines.join('\n')}\n---\n\n${body}\n`;
}
```

- [ ] **Step 2: Implement `site/scripts/extract.mjs`**

```js
import { writeFile, mkdir } from 'node:fs/promises';
import { dirname, join } from 'node:path';
import { existsSync } from 'node:fs';
import { urlToPath, stripDiviCruft, htmlToMarkdown, extractImageUrls, rewriteMediaUrls, toFrontmatter } from './lib/convert.mjs';

const BASE = process.env.WP_BASE || 'http://localhost:8080';
const API = `${BASE}/wp-json/wp/v2`;
const ROOT = new URL('..', import.meta.url).pathname;
const LANGS = ['th', 'en'];

async function getJSON(url) {
  const r = await fetch(url, { headers: { Accept: 'application/json' } });
  if (!r.ok) throw new Error(`${r.status} ${url}`);
  return r.json();
}

async function fetchAll(type, lang) {
  const out = [];
  for (let page = 1; ; page++) {
    const url = `${API}/${type}?lang=${lang}&per_page=100&page=${page}&_embed=1`;
    const r = await fetch(url, { headers: { Accept: 'application/json' } });
    if (r.status === 400) break;            // past last page
    if (!r.ok) throw new Error(`${r.status} ${url}`);
    const batch = await r.json();
    out.push(...batch);
    if (batch.length < 100) break;
  }
  return out;
}

async function downloadMedia(absUrl) {
  const rel = rewriteMediaUrls(`"${absUrl}"`).slice(1, -1);  // -> /media/....
  const dest = join(ROOT, 'public', rel);
  if (existsSync(dest)) return rel;
  const r = await fetch(absUrl);
  if (!r.ok) { console.warn(`media ${r.status}: ${absUrl}`); return rel; }
  await mkdir(dirname(dest), { recursive: true });
  await writeFile(dest, Buffer.from(await r.arrayBuffer()));
  return rel;
}

async function run() {
  let total = 0;
  for (const lang of LANGS) {
    const posts = await fetchAll('posts', lang);
    for (const p of posts) {
      const html = p.content?.rendered || '';
      for (const img of extractImageUrls(html)) await downloadMedia(img);
      let cover;
      const feat = p._embedded?.['wp:featuredmedia']?.[0]?.source_url;
      if (feat) cover = await downloadMedia(feat);
      const cleaned = rewriteMediaUrls(stripDiviCruft(html));
      const body = htmlToMarkdown(cleaned);
      const data = {
        title: p.title.rendered.replace(/&amp;/g, '&'),
        date: new Date(p.date),
        modified: new Date(p.modified),
        lang,
        slug: p.slug,
        path: urlToPath(p.link),
        categories: (p._embedded?.['wp:term']?.flat() || []).filter(t => t.taxonomy === 'category').map(t => t.name),
        tags: (p._embedded?.['wp:term']?.flat() || []).filter(t => t.taxonomy === 'post_tag').map(t => t.name),
        cover,
        excerpt: htmlToMarkdown(p.excerpt?.rendered || '').replace(/\n/g, ' ').trim() || undefined,
      };
      const file = join(ROOT, 'src/content/posts', lang, `${p.slug}.md`);
      await mkdir(dirname(file), { recursive: true });
      await writeFile(file, toFrontmatter(data, body));
      total++;
    }
    console.log(`posts[${lang}]: ${posts.length}`);
  }
  console.log(`TOTAL posts written: ${total}`);
}
run().catch(e => { console.error(e); process.exit(1); });
```
